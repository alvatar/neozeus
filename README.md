# neozeus

## Run

```bash
cargo run
```

## Keybindings

### Global
- `z` — open create-agent dialog
- `c` — open clone dialog for the selected clonable agent
- `Ctrl+Alt+R` — open reset confirmation dialog
- `Ctrl+K` — kill selected agent / selected owned tmux session
- `F10` — quit

### Focused agent / terminal
- `Enter` — open message composer
- `T` — open task editor
- `R` — rename focused agent
- `A` — open/disable Aegis for focused agent
- `N` — consume next task
- `P` — pause/unpause focused or selected agent
- `I` — show/hide the selected agent context box
- `Ctrl+T` — clear done tasks
- `Ctrl+V` — page up in terminal scrollback
- `Alt+V` — page down in terminal scrollback

### Dialogs / editors
- `Tab` / `Shift+Tab` — move focus
- `Enter` — confirm / submit
- `Esc` — cancel / close
